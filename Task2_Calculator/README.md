# Task 2 - Calculator

## Description
This project is a responsive Calculator developed as part of the CodeAlpha Frontend Development Internship. It performs basic arithmetic operations with a simple, clean, and user-friendly interface.

## Features
- Perform addition, subtraction, multiplication, and division
- Responsive design for desktop and mobile devices
- Clear (C) functionality
- User-friendly interface
- Fast and accurate calculations

## Technologies Used
- HTML5
- CSS3
- JavaScript
