# Task 1 - Image Gallery

## Description
This project is a responsive Image Gallery created as part of the CodeAlpha Frontend Development Internship. It provides an interactive gallery where users can browse images with a clean and modern interface.

## Features
- Responsive design for all screen sizes
- Image lightbox preview
- Previous and Next navigation
- Hover effects with smooth transitions
- Category-based image filtering
- Simple and user-friendly interface

## Technologies Used
- HTML5
- CSS3
- JavaScript
